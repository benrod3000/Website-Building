# Ben Rodriguez - Digital Marketing Portfolio

A personal website showcasing Ben Rodriguez's expertise in performance marketing, content systems, and brand positioning. The site features a clean, section-based design with generous spacing and strong visual hierarchy.

## Overview

This website highlights Ben's approach to building marketing systems designed for measurable outcomes, including conversion-focused campaign strategy, funnel design, and brand positioning through digital narrative.

## Sections

- **Hero**: Introduction to Ben's design philosophy for digital systems that convert attention into action.
- **Approach**: Key principles - Measured Systems, Conversion-First, Data-Driven, and Scalable.
- **About**: Overview of services including conversion-focused campaigns, funnel optimization, and brand positioning.
- **Capabilities**: Detailed breakdown of Performance Marketing, Content Systems, and Brand Positioning.
- **Strategy**: Step-by-step process: Positioning, Content System, Distribution, and Scaling.
- **Selected Work**: Case studies with real results, such as ROAS improvements and conversion rate increases.
- **Philosophy**: Core beliefs around no vanity metrics, speed + precision, and compounding growth.
- **Contact**: Call-to-action for collaborations and contact information.
- **Footer**: Copyright notice.

## Getting Started

1. Open `index.html` in your web browser to view the website.
2. To customize: Edit the content in `index.html`, update styles in `styles.css`, and modify scripts in `script.js`.
3. Replace `profile.jpg` with your own photo if needed.
4. The site is static and requires no build process.

## Files

- `index.html` — Main page structure with all sections and content.
- `styles.css` — Clean visual theme with large padding, generous spacing, and responsive design.
- `script.js` — JavaScript for dynamic elements, including updating the current year in the footer.
