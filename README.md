# Clayton Carroll Resume Website

A clean section-based resume framework with generous spacing and a strong visual hierarchy.

## Structure

This template includes:
- Hero Section
- Split Section
- Image Section
- Split Section
- Gallery Grid
- Call To Action
- Footer

## Getting Started

1. Open `index.html` in your web browser.
2. Replace placeholder text and the email address with your real contact details.
3. Update `profile.jpg` with your photo in the project folder.
4. Customize spacing, colors, and section content in `styles.css`.

## Files

- `index.html` — page structure and section layout
- `styles.css` — clean visual theme with large padding and generous spacing
- `script.js` — current year update
